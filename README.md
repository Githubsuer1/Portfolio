## 🚀 Setup & Usage

1. **Clone the Repository**
   ```sh
   git clone https://github.com/Githubsuer1/Portfolio_Site.git
   ```
2. **Navigate to the Project Folder**
   ```sh
   cd Portfolio_Site
   ```
3. **Open ********`index.html`******** in Your Browser**
   ```sh
   open index.html
   ```
   or simply drag & drop `index.html` into a browser.
